# @nxus/clientjs
A Browserify wrapper for Nxus applications to bundle client-side JS.

# Configuration

  ```
  "config": {
    "clientjs": {
      "watchify": true,
      "entries": {
        "assets/js/main.js": "public/js/bundle.js",
        "assets/js/minor.js": "public/js/other-bundle.js"
      }
    }
  }

  ```
